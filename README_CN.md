# 简单手写识别实现

利用 NN， CNN 等，实现简单的手写数字识别, 并使用 OpenCV 进行实时的检测。

## 神经网络实现

通过神经网络编程完成简单的数字手写识别，并测试自己手写的数字。

- [Neural_Network](https://nbviewer.jupyter.org/github/laugh12321/Handwriting-Recognition/blob/master/Neural_Network.ipynb)
